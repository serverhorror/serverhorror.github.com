Amazon yesterday published the source code of the Kindle DX. They have until now released three Kindle products.

  • Kindle
  • Kindle 2
  • Kindle DX

Over at blogkindle.com is a post covering some details about the original source code, but it seems that there hasn't been an update yet about the new Kindle DX source code. You can grab the source code for Kindle DX here. The page also lists the source code for the earlier product versions of the Kindle, so it may still be usefull to you even if you do not own a Kindle DX.

However the source code isn't the full code used in the Kindle DX (the same is true for the other downloads), it might still be interesting to skim over it or investigate the toolchain they used to provide the functionality of the Kindle DX.
